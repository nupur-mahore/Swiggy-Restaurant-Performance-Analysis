# Swiggy Sales Analysis Dashboard

## 📌 Project Overview
This project analyzes Swiggy sales data using Power BI to uncover key business insights such as revenue trends, city-wise performance, and order distribution patterns.

## 🎯 Objective
To perform data-driven analysis and identify performance metrics that can improve strategic decision-making in food delivery operations.

## 🛠 Tools & Technologies Used
- Power BI
- Python (Data Cleaning)
- CSV Dataset
- DAX (Data Analysis Expressions)

## 📊 Dashboard Insights
- Total Orders and Revenue Summary
- City-wise Sales Distribution
- Monthly Order Trends
- Top Performing Restaurants
- Average Delivery Time Analysis

## 📈 Key Findings
- High-demand cities contribute the majority of revenue.
- Seasonal trends impact order volume.
- A small group of restaurants generate significant sales.
